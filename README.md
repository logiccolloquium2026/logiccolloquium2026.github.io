# Logic Colloquium 2026 website

This is the Logic Colloquium 2026 website's repository.

Built using the **Jekyll framework**, combining previous work from another Swansea University event [website](https://swansea-theory.github.io/conferences/CCC2025/index.html) and Minima (the default Jekyll theme).
