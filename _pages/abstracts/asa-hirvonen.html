Speaker: Åsa Hirvonen

Title

Ultraproduct approximations in metric model theory.

Abstract

Continuous logic offers an approach to metric structures that in many aspects resembles classical first order model theory. As for ultraproducts, both Los’s theorem and the Keisler-Shelah theorem have analogues in a continuous logic setting. However, as continuous logic is positive, we get some new features when building ultraproducts. E.g., the formulas we wish to hold in our ultraproduct model need not be true in any of the constituent models.

In this talk I will explore some of the possibilities ultraproducts offer as a means for approximation. In physics there is a tradition of approximating quantum mechanical systems via finite dimensional spaces, and as an example I will sketch how the properties of metric ultraproducts can be used to justify such approximations. This part of the talk is based on joint work with Tapani Hyttinen.
